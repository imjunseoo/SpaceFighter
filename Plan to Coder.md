# [작업 지시서] 메인 로비 상호작용 캐릭터 'Miku' 구현

본 문서는 메인 로비에 생동감을 불어넣고 플레이어와 상호작용할 수 있는 전술 보조 AI 'Miku'를 구현하기 위한 작업 지시서입니다.

---

## 1. Miku 캐릭터 배치 및 애니메이션 (`index.html`, `css/style.css`)
- **캐릭터 이미지**: `Image/Miku_Idle.png`를 사용합니다.
- **배치**: `main-ui-overlay` 내부의 `<main>` 섹션(로비 메인 영역)에 배치합니다.
- **기본 애니메이션**: 
    - 기존 `aria-float` 클래스를 적용하여 위아래로 부생하는 연출을 유지합니다.
    - **자율 이동(Roaming)**: CSS `transition`을 활용하여 위치(left, top) 변경 시 부드럽게 이동하도록 설정합니다.

## 2. 자율 이동(Roaming) 로직 구현 (`js/engine.js`)
- **이동 범위**: 로비의 메인 콘텐츠 영역(`<main>`) 내부로 한정합니다. 사이드바(aside)나 헤더(header) 영역을 침범하지 않도록 경계값을 계산해야 합니다.
- **이동 주기**: 10~15초마다 랜덤한 좌표로 위치를 변경합니다.
- **상태 관리**: 말풍선이 활성화되어 대화 중일 때는 이동을 일시 정지합니다.

## 3. 상호작용 및 피드백 시스템 (`js/engine.js`)
- **클릭 이벤트(Click)**:
    - **시각 피드백**: 클릭 시 Miku 캐릭터가 위로 살짝 튀어오르는 점프(Jump) 효과와 이미지의 밝기가 일시적으로 높아지는 발광(Glow) 효과를 적용합니다.
    - **음향 피드백**: 기존 튜토리얼 음원(`Sound/Mikudayo sound.mp3`)을 짧게 활용하거나 적절한 효과음을 재생합니다.
- **말풍선(Speech Bubble)**:
    - Miku 머리 위에 레이아웃이 나타나며, 아래의 대사 풀에서 하나를 랜덤하게 선택하여 타이핑 효과와 함께 출력합니다.
    - 대사 출력 완료 후 5초 뒤에 자동으로 페이드아웃 처리합니다.

## 4. 대사 풀 (Dialogue Pool)
1. "시스템 정상 가동 중. 언제든 출격 가능해, 파일럿!"
2. "적들의 신호가 점점 강해지고 있어. 조심해야 해."
3. "15레벨이 되면 첫 번째 보스가 나타날 거야. 준비는 됐어?"
4. "무기 업그레이드는 5스택마다 진화한다는 사실, 잊지 마!"
5. "잠깐 쉬는 것도 전략이야. 내가 감시하고 있을게."
6. "오늘의 전술 데이터 분석 완료. 성능 향상을 기대해도 좋아."
7. "가끔씩 시스템 어딘가에서 대파가 등장한다는 소문이 있던데...."

## 5. 기존 UI 연동
- **게임 시작 시**: `MISSION_START` 버튼 클릭 시 Miku 캐릭터와 말풍선은 즉시 숨겨져야 하며, 게임 로직에 간섭하지 않아야 합니다.
